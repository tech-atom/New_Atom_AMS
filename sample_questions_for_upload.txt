Question 1 [MCQ]:
What is the primary function of the CPU in a computer?
A) Store data permanently
B) Process instructions and perform calculations
C) Display graphics on screen
D) Connect to the internet
Correct Answer: B
Explanation: The CPU (Central Processing Unit) is the brain of the computer that executes instructions and performs calculations.

---

Question 2 [MCQ]:
Which of the following is NOT a programming language?
A) Python
B) HTML
C) Java
D) Photoshop
Correct Answer: D
Explanation: Photoshop is image editing software, not a programming language.

---

Question 3 [MCQ]:
What does SQL stand for?
A) Standard Query Language
B) Structured Query Language
C) Simple Question Language
D) System Quality Language
Correct Answer: B
Explanation: SQL (Structured Query Language) is used for managing and querying databases.

---

Question 4 [MCQ]:
In Python, which symbol is used for comments?
A) //
B) /* */
C) #
D) --
Correct Answer: C
Explanation: Python uses the hash symbol (#) for single-line comments.

---

Question 5 [MCQ]:
What is the binary equivalent of decimal number 10?
A) 1010
B) 1100
C) 1001
D) 1111
Correct Answer: A
Explanation: 10 in decimal = 1010 in binary (8 + 2 = 10)

Question 6 [TRUE/FALSE]:

Options: True / False
Correct Answer: True
Explanation: HTTP is indeed HyperText Transfer Protocol, used for transmitting web pages.

---

Question 7 [TRUE/FALSE]:
An IP address is a unique identifier for devices on a network.
Options: True / False
Correct Answer: True
Explanation: Every device on a network has a unique IP address for identification and communication.

---

Question 8 [TRUE/FALSE]:
JavaScript and Java are the same programming language.
Options: True / False
Correct Answer: False
Explanation: Despite similar names, JavaScript and Java are completely different languages with different purposes.

---

Question 9 [TRUE/FALSE]:
The binary number system uses only 0 and 1.
Options: True / False
Correct Answer: True
Explanation: Binary is a base-2 number system consisting only of digits 0 and 1.

Question 10 [TRUE/FALSE]:
HTTPS is less secure than HTTP.
Options: True / False
Correct Answer: False
Explanation: HTTPS is MORE secure than HTTP as it encrypts data transmission using SSL/TLS.
